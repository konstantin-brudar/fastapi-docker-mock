# Заглушка на FastAPI в Docker

Заглушка для web приложения на FastAPI в Docker контейнере.

## Запуск приложения

Для запуска контейнера с приложением нужно выполнить команду:
```bash
sudo docker compose up -d
```

Для проверки работы приложения откройте в браузере страницу по адресу:
```
http://127.0.0.1:8000
```

Документация Swagger:
```
http://127.0.0.1:8000/docs
```

Для остановки контейнера с приложением нужно выполнить команду:
```bash
sudo docker compose down
```
